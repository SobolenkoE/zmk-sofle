# ZMK Sofle Firmware

Персональная прошивка для сплит-клавиатуры Sofle на базе ZMK.

## Источники

- Оригинальный репозиторий: https://github.com/a741725193/zmk-sofle
- Форк: https://github.com/SobolenkoE/zmk-sofle

## Особенности прошивки

- ZMK 3.5
- Поддержка дисплея nice_view
- Поддержка тачпада
- Настройки питания и сна
- Кастомные слои и макросы

## Сборка прошивки

Требования:
- west
- zmk-dev-container

Команда сборки:
\`\`\`bash
west build -b nice_nano_v2 -d build/left -- -DSHIELD=eyelash_sofle_left -DZMK_CONFIG="config"
west build -b nice_nano_v2 -d build/right -- -DSHIELD=eyelash_sofle_right -DZMK_CONFIG="config"
\`\`\`

Файлы прошивки будут находиться в:
- \`build/left/zephyr/zmk.uf2\` (левая сторона)
- \`build/right/zephyr/zmk.uf2\` (правая сторона)

## Установка прошивки

1. Нажмите на кнопку сброса (reset) на клавиатуре
2. Клавиатура появится как USB-накопитель
3. Скопируйте файл \`zmk.uf2\` на этот диск
4. Клавиатура перезагрузится с новой прошивкой

## Карты слоёв

Базовый слой (Layer 0):
- Стандартная раскладка QWERTY
- Стрелки на левой и правой частях клавиатуры

Слой 1:
- Функциональные клавиши
- Управление тачпадом
- Настройки RGB

Слой 2:
- Настройки Bluetooth
- Переключение между USB/BLE
- Функции сброса

## Управление питанием

- **Soft Off**: Q + S + Z (удерживать 2 секунды)
- Выход из Soft Off: нажатие кнопки сброса
- Автоматический переход в спящий режим: 1 час бездействия
